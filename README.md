# Polar Chart Form 
**Contributors:** chasing6  

This plugin adds a polar chart with a custom form to control it.


## Description 

This plugin adds a polar chart with a custom form to control it.

To add a the chart and form, use the shortcode `[polar_chart]` to any post or page.
